# Auspicious Time — Reference

## ฤกษ์ยามไทย (Thai Auspicious Times)

### ช่วงเวลาตามโหราศาสตร์ไทย
- ราศีที่ดาวเคลื่อนผ่าน = ฤกษ์ดีสำหรับกิจกรรมนั้น
- ดาวอาทิตย์ในราศี = วันดีทั่วไป
- ดาวศุกร์ในราศีที่ดี = วันดีเรื่องความรัก

### ราหูกาล (Rahu Kaal)
- ช่วงเวลาที่ควรหลีกเลี่ยงเริ่มสิ่งใหม่
- เปลี่ยนไปทุกวัน (ประมาณ 1.5 ชม./วัน)
- วันจันทร์: 07:30-09:00
- วันอังคาร: 15:00-16:30
- วันพุธ: 12:00-13:30
- วันพฤหัส: 13:30-15:00
- วันศุกร์: 10:30-12:00
- วันเสาร์: 09:00-10:30
- วันอาทิตย์: 16:30-18:00

### ฤกษ์ตามวัน
- วันจันทร์: เหมาะขอเลขเด็ด ทำบุญ
- วันอังคาร: หลีกเลี่ยงเริ่มโปรเจกต์ใหญ่
- วันพุธ: เหมาะเซ็นสัญญา ต่อรอง
- วันพฤหัส: เหมาะลงทุน เปิดกิจการ
- วันศุกร์: เหมาะนัดคู่ เสนอตัว
- วันเสาร์: เหมาะทำบุญ ปล่อยปล่อย
- วันอาทิตย์: เหมาะขอความก้าวหน้า

### ดาวพลัส (Benefic Transits)
- ดาวพฤหัสบดีทับลัคนา = โชคดี
- ดาวศุกร์ทับดาวอาทิตย์ = เสน่ห์ดี
- ดาวพุธทับดาวเกิด = ความคิดดี
