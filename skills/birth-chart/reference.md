# Birth Chart — Reference

## การคำนวณลัคนา (Ascendant)
ลัคนาเปลี่ยนทุก 2 ชั่วโมง ขึ้นกับเวลาเกิด:

### คำนวณคร่าวๆ จากเวลาเกิด
- 06:00-08:00: ลัคนาใกล้เคียงราศีเมษ
- 08:00-10:00: ลัคนาใกล้เคียงราศีพฤษภ
- 10:00-12:00: ลัคนาใกล้เคียงราศีเมถุน
- 12:00-14:00: ลัคนาใกล้เคียงราศีกรกฎ
- 14:00-16:00: ลัคนาใกล้เคียงราศีสิงห์
- 16:00-18:00: ลัคนาใกล้เคียงราศีกันย์
- 18:00-20:00: ลัคนาใกล้เคียงราศีตุลย์
- 20:00-22:00: ลัคนาใกล้เคียงราศีพิจิก
- 22:00-00:00: ลัคนาใกล้เคียงราศีธนู
- 00:00-02:00: ลัคนาใกล้เคียงราศีมังกร
- 02:00-04:00: ลัคนาใกล้เคียงราศีกุมภ์
- 04:00-06:00: ลัคนาใกล้เคียงราศีมีน

*หมายเหตุ: นี่เป็นการคำนวณคร่าวๆ ไม่ได้คำนึงถึง latitude/longitude

## ดาวประจำราศี (Ruling Planets)
- เมษ: อังคาร
- พฤษภ: ศุกร์
- เมถุน: พุธ
- กรกฎ: จันทร์
- สิงห์: อาทิตย์
- กันย์: พุธ
- ตุลย์: ศุกร์
- พิจิก: อังคาร (ร่วมกับมังกร/เกตุ)
- ธนู: พฤหัสบดี
- มังกร: เสาร์
- กุมภ์: เสาร์ (ร่วมกับราหู)
- มีน: พฤหัสบดี
