# AI Chat — Reference

## ระบบโหราศาสตร์ไทย

### บ้านทั้ง 12 (Houses)
1. ตนุ (ตัวเอง รูปร่าง)
2. ธนุ (เงิน ทรัพย์สิน)
3. สหัช (พี่น้อง การสื่อสาร)
4. พันธุ (แม่ บ้าน ที่ดิน)
5. ปุตตะ (ลูก ความรัก การศึกษา)
6. อริ (ศัตรู โรคภัย)
7. กามะ (คู่ครอง การแต่งงาน)
8. มรณะ (อายุ มรดก ความลับ)
9. ภัคยะ (บิดา โชคลาภ ศาสนา)
10. กรม (อาชีพ ชื่อเสียง)
11. ลาภะ (รายได้ เพื่อน)
12. วินาศ (ค่าใช้จ่าย การสูญเสีย)

### ดาวเคราะห์ในโหราศาสตร์ไทย
- อาทิตย์ (Surya): ตัวตน ผู้นำ
- จันทร์ (Chandra): จิตใจ ความรู้สึก
- อังคาร (Mangala): พลัง ความก้าวร้าว
- พุธ (Budha): ปัญญา การสื่อสาร
- พฤหัสบดี (Brihaspati): โชคลาภ ครู
- ศุกร์ (Shukra): ความรัก ความงาม
- เสาร์ (Shani): วินัย อุปสรรค บทเรียน
- ราหู (Rahu): ความทะเยอทะยาน สิ่งลึกลับ
- เกตุ (Ketu): จิตวิญญาณ การปล่อยวาง

### ธาตุ (Elements)
- ไฟ: เมษ สิงห์ ธนู
- ดิน: พฤษภ กันย์ มังกร
- ลม: เมถุน ตุลย์ กุมภ์
- น้ำ: กรกฎ พิจิก มีน
