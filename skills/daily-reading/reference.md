# Daily Reading — Reference

## ดาวเคราะห์และความหมาย
- อาทิตย์: ตัวตน บุคลิก ผู้นำ
- จันทร์: อารมณ์ ความรู้สึก ครอบครัว
- อังคาร: พลังงาน การต่อสู้ กล้าหาญ
- พุธ: การสื่อสาร ความคิด ธุรกิจ
- พฤหัสบดี: โชคลาภ การเงิน การศึกษา
- ศุกร์: ความรัก ความงาม ศิลปะ
- เสาร์: ความรับผิดชอบ วินัย อุปสรรค

## ลัคนา (Ascendant) และความหมาย
- เมษ: มั่นใจ กล้าหาญ เป็นผู้นำ
- พฤษภ: มั่นคง อดทน รักสวยงาม
- เมถุน: เก่งการสื่อสาร ฉลาด ยืดหยุ่น
- กรกฎ: รักครอบครัว อ่อนไหว ปกป้องคนรอบข้าง
- สิงห์: ภูมิใจ ใจกว้าง ชอบเป็นจุดศูนย์กลาง
- กันย์: รักสะอาด พิถีพิถัน ช่วยเหลือผู้อื่น
- ตุลย์: รักความยุติธรรม สมดุล รักสันติ
- พิจิก: ลึกลับ เข้มแข็ง มีสัญชาตญาณ
- ธนู: รักอิสระ ชอบผจญภัย มองโลกในแง่ดี
- มังกร: ทะเยอทะยาน มีวินัย รับผิดชอบ
- กุมภ์: เป็นตัวของตัวเอง มนุษยนิยม คิดแปลก
- มีน: จินตนาการสูง ไหวลื่น มีความรู้สึกละเอียด

## วันดี/วันระวัง (ตัวอย่าง)
- วันจันทร์: เหมาะเรื่องครอบครัว อสังหาริมทรัพย์
- วันอังคาร: ระวังเรื่องความขัดแย้ง
- วันพุธ: เหมาะเรื่องธุรกิจ สัญญา
- วันพฤหัส: เหมาะเรื่องการเงิน การลงทุน
- วันศุกร์: เหมาะเรื่องความรัก ความสัมพันธ์
- วันเสาร์: ระวังเรื่องอุปสรรค ความล่าช้า
- วันอาทิตย์: เหมาะเรื่องตำแหน่ง ชื่อเสียง
