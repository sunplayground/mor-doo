# Dream Interpretation — Reference

## พจนานุกรมฝันไทย (ฉบับย่อ)

### สัตว์
- งู: เรื่องเสน่ห์ หรือศัตรูซ่อนเร้น
- ปลา: โชคลาภ เงินทอง
- นก: ข่าวดี การเดินทาง
- แมว: ศัตรูหรือคนที่ไม่ซื่อสัตย์
- หมา: มิตรภาพ หรือคนที่คอยปกป้อง
- ช้าง: อำนาจ ความสำเร็จใหญ่
- เสือ: อุปสรรค ศัตรูที่แข็งแกร่ง
- มด/ผึ้ง: ความขยัน ผลงานที่ตามมา

### สิ่งของ/สถานที่
- ทอง/เงิน: โชคลาภ
- บ้าน: ตัวเอง ครอบครัว สถานะ
- น้ำ: การเงิน อารมณ์
- ไฟ: ความโกรธ หรือความสว่าง
- ดอกไม้: ความรัก ความสุข
- วัด: สิ่งศักดิ์สิทธิ์ คุ้มครอง
- ต้นไม้: การเจริญเติบโต
- เสา: การสนับสนุน ความมั่นคง

### บุคคล
- พระ: โชคลาภ สิ่งดีงาม
- คนตาย: การเปลี่ยนแปลง หรือข่าวจากคนไกล
- เด็ก: เรื่องใหม่ ความสดใส
- แฟนเก่า: ความทรงจำ หรือเรื่องที่ยังไม่จบ
- คนแปลกหน้า: โอกาสใหม่
