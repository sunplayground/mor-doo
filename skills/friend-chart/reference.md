# Friend Chart — Reference

## ความเข้ากันของราศี

### คู่ราศีที่เข้ากันดี (Trine - ธาตุเดียวกัน)
- ไฟ: เมษ-สิงห์, สิงห์-ธนู, เมษ-ธนู
- ดิน: พฤษภ-กันย์, กันย์-มังกร, พฤษภ-มังกร
- ลม: เมถุน-ตุลย์, ตุลย์-กุมภ์, เมถุน-กุมภ์
- น้ำ: กรกฎ-พิจิก, พิจิก-มีน, กรกฎ-มีน

### คู่ราศีที่เสริมกัน (Sextile)
- เมษ-เมถุน, เมษ-กุมภ์
- พฤษภ-กรกฎ, พฤษภ-พิจิก
- เมถุน-เมษ, เมถุน-ธนู
- กรกฎ-พฤษภ, กรกฎ-พฤษภ
- สิงห์-ตุลย์, สิงห์-มังกร
- กันย์-มีน, กันย์-พิจิก

### คู่ราศีที่ชนกัน (Square/Opposition)
- ราศีตรงข้าม: เมษ-ตุลย์, พฤษภ-พิจิก, เมถุน-ธนู, กรกฎ-มังกร, สิงห์-กุมภ์, กันย์-มีน
- ราศีที่ทำมุมฉาก: ห่างกัน 3 ราศี

### 4 มิติการวิเคราะห์
1. **Chemistry**: ดาวศุกร์และจันทร์ของทั้งสองคน
2. **Communication**: ดาวพุธของทั้งสองคน
3. **Friction**: ดาวอังคารและเสาร์ จุดที่ขัดแย้ง
4. **Longevity**: ดาวพฤหัสและเสาร์ ความยั่งยืน
