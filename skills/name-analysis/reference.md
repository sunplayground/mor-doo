# Name Analysis — Reference

## การนับขีดตัวอักษรไทย

### สระ (เสียง)
- อะ อา อิ อี อุ อู เอ แอ โอ ออ เอา ไอ ใอ ฯลฯ: นับ 1 ขีด

### พยัญชนะ (ตัวอักษร)
จำนวนขีดโดยประมาณ:
- ก ข ค ง: 1-2 ขีด
- จ ฉ ช ซ ฌ ญ: 1-3 ขีด
- ฎ ฏ ฐ ฑ ฒ ณ: 2-4 ขีด
- ด ต ถ ท ธ น: 1-2 ขีด
- บ ป ผ ฝ พ ฟ ภ ม: 1-3 ขีด
- ย ร ล ว: 1-2 ขีด
- ศ ษ ส ห ฬ อ ฮ: 1-3 ขีด

### ตัวเลขมงคลจากขีด
- รวมขีดทั้งชื่อ ลดเหลือเลข 1 หลัก
- เลข 1: เหมาะผู้นำ
- เลข 3: เหมาะศิลปิน
- เลข 5: เหมาะนักธุรกิจ
- เลข 6: เหมาะครอบครัว
- เลข 8: เหมาะการเงิน

### หลักการตั้งชื่อมงคล
1. ขีดรวมเป็นมงคล (8, 9, 11, 13, 15, 16, 21, 23, 25, 31, 32, 33, 35, 37)
2. เสียงดี ออกเสียงง่าย
3. ไม่มีความหมายไม่ดี
4. เข้ากับธาตุของผู้ใช้
