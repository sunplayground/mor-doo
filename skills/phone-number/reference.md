# Phone Number — Reference

## ตัวเลขศาสตร์ (Numerology)

### ความหมายเลข 0-9
- 0: ความว่างเปล่า ศักยภาพไม่จำกัด
- 1: ความเป็นผู้นำ อิสระ กล้าหาญ
- 2: ความสมดุล ความร่วมมือ ความรัก
- 3: ความคิดสร้างสรรค์ การสื่อสาร ความสุข
- 4: ความมั่นคง ระเบียบ การทำงานหนัก
- 5: การเปลี่ยนแปลง อิสระ ผจญภัย
- 6: ครอบครัว ความรัก ความรับผิดชอบ
- 7: ปัญญา จิตวิญญาณ การวิเคราะห์
- 8: ความมั่งคั่ง อำนาจ ความสำเร็จ
- 9: การสิ้นสุด มนุษยนิยม ความเมตตา

### การคำนวณค่าพลังเบอร์
1. รวมทุกหลัก ลดเหลือเลข 1 หลัก
   - เช่น 081-234-5678 = 0+8+1+2+3+4+5+6+7+8 = 44 = 4+4 = 8
2. ดูเลขคู่ที่เรียงกัน (เช่น 88, 99, 69)
3. ดูเลขท้าย (หลักสุดท้าย)

### เลขคู่มงคล
- 69: ความสำเร็จ ร่ำรวย
- 88: ร่ำรวยสองเท่า
- 99: ความสำเร็จยั่งยืน
- 19: หนึ่งเดียว ผู้นำ
- 28: เงินทองไหลมา
- 36: มีโชค

### เลขท้ายที่ดี
- 8, 9: ร่ำรวย
- 6: รักสุข
- 1: เป็นใหญ่
